# 队列

队列是一种特殊的列表，队列只能在队尾插入元素，在队首删除元素。队列用于存储按顺序排列的数据，先进先出。

## 参数

接收的参数为数组类型。

## 属性及方法

栈的抽象数据类型定义

| 列表 | 属性或方法 | 描述 |
|----|----|----|
| enqueue | 方法 | 向队尾加入一个元素 |
| dequeue | 方法 | 删除队首的元素 |
| front | 方法 | 读取队首元素 |
| back | 方法 | 读取队尾元素 |
| toString | 方法 | 返回队列的字符串形式 |
| clear | 方法 | 清空队列 |
| length | 方法 | 返回队列内元素的个数 |
| empty | 方法 | 队列是否为空 |
